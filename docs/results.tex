\section*{Setup}
\hd{Data.} IoT-23 scenario 34-1 pcap (120.5\,MB, 233{,}865 packets) $\rightarrow$ \textbf{4{,}662 flows}, each the first $K{=}32$ packets $\times$ 13 payload-free features; labels joined from the Zeek \texttt{conn.log.labeled} by 5-tuple. Classes: Mirai 4055, Benign 290, DDoS 211, Recon 106. Split 70/15/15 (test 700).\\
\hd{Model.} Mamba encoder ($d{=}128$, 4 layers) $+$ softmax classifier $+$ one-class deep-SVDD anomaly head.\\
\hd{Run.} 3 stages $\times$ 8 epochs, CPU (Apple M3 Pro). Extraction $\approx$\,1.5\,s; \textbf{training 659\,s ($\approx$11\,min)}; inference \textbf{5.1\,ms/flow}.
